  1. Go to Setup → App Manager → New Connected App

  2. App Name: Sales Dashboard. Enable OAuth Settings. Callback URL: http://localhost

  3. Add scopes: api and refresh_token. Save. Wait 2–10 min.

  4. Copy Consumer Key and Consumer Secret — paste them in the form →

  5. Get your Security Token from My Settings → Reset Security Token. Append it directly to your password.
